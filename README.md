# moyan-utils

##### 记录一些复杂业务场景使用的 js 类库。

> 欢迎有想法, 热衷技术的小伙伴一起共建啦! 致力于打造最好用的前端工具库, 您可以提 issue 或者 pr 来共建项目，谢谢.

## Installation

```
npm i moyan-utils
```

## All Function List

- 常用判断函数

  - isEmpty - 判断空对象
  - isJSDataType - 判断数据类型

- 数据结构相关

  - chunk - 数组拆分
  - get - 获取数据的值
  - flatten - 数组扁平化
  - cloneDeep - 数据深拷贝
  - omit - 生成忽略属性的新对象
  - pick - 生成选中属性的新对象
  - getJSDataType - 获取数据类型
  - formatNumber - 数字千分位展示
  - pickBy - 生成经 predicate 判断为真值的属性的对象

- 高级函数

  - memoize - 记忆函数

- 数学计算

  - getRandomColor - 获取随机颜色
